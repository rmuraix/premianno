# PremiAnnoとは？

PremiAnnoは、Adobe Premiere Pro用のUXPベースのプラグインで、タイムライン上で直接、時間同期アノテーションを可能にします。AIリサーチプロジェクトやメディア分析ワークフロー向けに設計されており、研究者やクリエイターが編集環境を離れることなく、構造化されたメタデータ、タグ、コメントを追加できます。

## なぜPremiAnnoなのか？

Premiere Proにアノテーション機能を統合することで、PremiAnnoはクリエイティブな編集と機械学習、ビデオ理解、マルチモーダルAIリサーチのためのデータ収集との間のギャップを埋めます。

### ユースケース

- **AIリサーチ**: ビデオ理解と機械学習モデルのためのラベル付きデータセットを収集
- **メディア分析**: コンテンツ分析とリサーチのための構造化されたメタデータを追加
- **データセット作成**: AIモデルのトレーニング用にアノテーション付きビデオデータセットを構築
- **ワークフロー統合**: 既存の編集ワークフローにアノテーションをシームレスに統合

## 主な機能

- **時間ベースのアノテーション**: 特定のタイムスタンプやクリップ範囲にノート、ラベル、メタデータを添付
- **カスタムタグセット**: 独自のアノテーションスキーマ（感情、アクション、シーンなど）を定義
- **タイムライン統合**: アノテーションはクリップの移動や編集に追従
- **エクスポートオプション**: アノテーションをJSON/CSVとしてエクスポートし、下流のAIタスクに活用

## 仕組み

PremiAnnoはAdobe Premiere ProのUXP（Unified Extensibility Platform）と直接統合され、アプリケーション内にネイティブパネルを提供します。これにより、以下のことが可能になります：

1. クリップまたはタイムライン位置を選択
2. カスタムタグとメタデータを使用してアノテーションを追加
3. リアルタイムでアノテーションを表示・管理
4. さらなる分析のためにアノテーション付きデータをエクスポート
