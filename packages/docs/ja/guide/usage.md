# 使い方ガイド

Adobe Premiere ProでPremiAnnoを使用してビデオプロジェクトにアノテーションを付ける方法を学びます。

## 概要

PremiAnnoは、ビデオタイムラインに時間同期アノテーションを追加するための効率的なインターフェースを提供します。このガイドでは、主な機能とワークフローについて説明します。

## 基本的なワークフロー

### 1. PremiAnnoを開く

1. Adobe Premiere Proを起動
2. プロジェクトを開くまたは作成
3. **ウィンドウ > 拡張機能 > PremiAnno**に移動
4. PremiAnnoパネルがワークスペースにドッキングされます

### 2. アノテーションを作成する

タイムライン上の特定のポイントまたは範囲にアノテーションを追加できます：

1. **再生ヘッドを配置**: 再生ヘッドを目的のタイムスタンプに移動
2. **コンテンツを選択**（オプション）: タイムライン上のクリップまたは範囲を選択
3. **アノテーションを追加**: PremiAnnoパネルを使用して新しいアノテーションを作成
4. **詳細を追加**: アノテーションフィールド（タグ、ノート、メタデータ）を入力

### 3. アノテーションを管理する

作成したアノテーションは以下のことができます：

- **編集**: タグ、ノート、メタデータを変更
- **移動**: 編集の変更に合わせてタイミングを調整
- **削除**: 不要になったアノテーションを削除
- **フィルタリング**: 特定の種類のアノテーションを表示

### 4. データをエクスポートする

アノテーション付きデータを使用する準備ができたら：

1. PremiAnnoパネルで**エクスポート**をクリック
2. フォーマット（JSONまたはCSV）を選択
3. 出力場所を選択
4. アノテーション付きデータセットが分析用に準備完了

## 高度な機能

### カスタムタグセット

独自のアノテーションスキーマを定義：

1. **タグ設定**パネルを開く
2. カスタムカテゴリを作成（例：感情、アクション、オブジェクト）
3. 各カテゴリの下に特定のタグを追加
4. プロジェクト全体でこれらのタグを一貫して使用

### タイムライン同期

PremiAnnoはアノテーションの同期を維持します：

- クリップを並べ替えると、アノテーションもクリップと一緒に移動
- クリップをトリミングすると、タイミングが更新
- 編集を通してマーカーが正確に保たれる

### バッチ操作

複数のアノテーションを効率的に処理：

- 一括編集のために複数のアノテーションを選択
- コンテンツの範囲にタグを適用
- アノテーションのフィルタリングされたサブセットをエクスポート

## ベストプラクティス

### 整理

- タグには一貫した命名規則を使用
- アノテーションを開始する前にスキーマを作成
- アノテーションカテゴリを文書化

### 効率

- キーボードショートカットを使用（利用可能な場合）
- 一般的なアノテーションタイプのテンプレートを作成
- 作業を保存するために定期的にエクスポート

### コラボレーション

- タグスキーマをチームメンバーと共有
- 一貫したアノテーションフォーマットを使用
- 複数のエディターからデータセットをエクスポートしてマージ

## ヒントとコツ

- **頻繁に保存**: アノテーションを定期的にエクスポート
- **プレビューモード**: Premiereの再生を使用してアノテーションのタイミングを確認
- **タグの階層**: タグを論理的なカテゴリに整理
- **メタデータフィールド**: プロジェクト固有のデータにカスタムフィールドを使用

## トラブルシューティング

### アノテーションが保存されない

- プロジェクトファイルが書き込み可能であることを確認
- 利用可能なディスク容量を確認
- データの整合性を確認するためにエクスポートを試行

### タイミングの問題

- シーケンス設定がソースメディアと一致することを確認
- タイミングに影響する可能性のあるギャップやカットを確認
- タイムラインが大幅に変更された場合はアノテーションを再同期

### パフォーマンス

- パフォーマンスを向上させるために未使用のパネルを閉じる
- 古いアノテーションをエクスポートしてアーカイブ
- プラグインが遅くなった場合はPremiere Proを再起動
